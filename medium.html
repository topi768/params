Задание 1

монотонность

Найдите наименьшее значение параметра а, при котором функция является неубывающей на всей числовой прямой

$$ y = 9 + 7x - 3| ax + 2 | + | ax + 5 | + | x + 1 | $$

Данная функция определена на всей числовой прямой и при любом значении параметра а является кусочно-линейной. Точки

$$ -1, -\frac{ 2 } { a }, -\frac{ 5 } { a } $$

разбивают числовую прямую на промежутки, на каждом из которых функция является линейной. Рассмотрим случай a < 0. Тогда

$$ -1 < -\frac{ 2 } { a } < -\frac{ 5 } { a } $$

. При раскрытие модулей получаем:

$$ если \ x < -1, то \ y = 9 + 7x - 3ax - 6 + ax + 5 - x - 1 = (6 - 2a)x + 7$$ $$ если \ -1 \leq x < -\frac{2}{a} , то \ y = 9 + 7x - 3ax - 6 + ax + 5 + x + 1 = (8 - 2a)x + 9$$ $$ если \ -\frac{2}{a} \leq x < -\frac{5}{a} , то \ y = 9 + 7x + 3ax + 6 + ax + 5 + x + 1 = (8 - 4a)x + 21$$ $$ если \ x \geq -\frac{5}{a} , то \ y = 9 + 7x + 3ax + 6 - ax - 5 + x + 1 = (8 + 2a)x + 11$$

Для того чтобы функция не убывала, необходимо и достаточно, чтобы все полученные значения угловых коэффициентов прямых были неотрицательными. Следовательно,

$$ \begin{equation*} \begin{cases} a < 0 \\ 6 - 2a \geq 0 \\ 8 - 2a \geq 0 \\ 8 - 4a \geq 0 \\ 8 + 2a \geq 0 \end{cases} \end{equation*} $$

получаем:

$$ -2 \leq a < 0 $$

Отсюда получаем наименьшее значение параметра, равное -2. Так как по условию задачи нужно найти наименьшее значение а, то случай а > 0 рассматривать не нужно.
